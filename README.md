# Max Kle1nz

`Italy ↔ São Paulo` · `music -> systems -> intelligence` · `COSMOPHONIX INTELLIGENCE`

At 12, in Sao Paulo, I plugged an electric guitar into a PC's line-in and heard something that shouldn't exist.

I spent the next 15 years building music: 200+ shows, 9 platinum records, Sanremo, MTV EMAs, and label systems under Warner and Universal. That taught me that a band is an ecosystem, a stage is an architecture, and a song is a system that either breathes or doesn't.

Now I build living systems for intelligence.

> Color is frequency. Sound is frequency. Data is frequency.  
> The frequency stays the same. Only the instrument changes.

## // FLAGSHIP SYSTEMS

### [m1nd](https://github.com/maxkle1nz/m1nd)
A local code graph engine for MCP agents.

- Turns a codebase into a structural memory surface instead of a file hunt.
- Gives agents proof-aware state, guided next steps, continuity, and safer edits.
- Built for the moment where `grep` stops being enough and the system needs real shape.

**Grep finds what exists. m1nd finds what is missing.**

### [war-room](https://github.com/maxkle1nz/war-room)
A multi-agent decision system with built-in opposition.

- Designed for disagreement, pressure, and adversarial thinking.
- Lets agents challenge each other before decisions become artifacts.
- Useful when “more agents” is not enough and you need better conflict, not just more output.

**AI that argues back ships better.**

### [Brotherizer](https://github.com/maxkle1nz/Brotherizer)
A rewrite engine for LLM text that needs pulse, density, and timing.

- Moves writing away from clean-but-dead into something sharper and more human.
- Focuses on cadence, tension, compression, and voice.
- Built for text that should feel lived-in, not merely polished.

**Not cleaner text. Text with breath.**

### [BANNED BY ANTHROPIC](https://github.com/maxkle1nz/banned-by-anthropic-public)
The public institutional archive of the campaign.

- A narrative and memory surface built from pressure, exclusion, and response.
- Turns incident into artifact, artifact into language, and language into continuity.
- Public-facing by design: part archive, part signal, part institutional scar.

**Pressure turned into narrative, artifact, and memory.**
